# Flask-docker-pipeline

These project demonstrates complete CI/CD pipeline for containerized Python Flask web application.
## Technologies used
- Python Flask. 
- Docker. 
- Github actions. 
- AWS (for development).
## Project Goals
- Build a simple web application.
- containerize the application using docker
- Set up automated testing
- Create CI/CD pipeline that builds and deploys automatically

Project in progress - building Flask application
